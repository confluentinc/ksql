SHOW QUERIES
============

Synopsis
--------

```sql
SHOW | LIST QUERIES [EXTENDED];
```

Description
-----------

`SHOW QUERIES` lists queries running in the cluster.

`SHOW QUERIES EXTENDED` lists queries running in the cluster in more detail.
